A library for encoding and decoding
[QOI](https://en.wikipedia.org/wiki/QOI_(image_format)) images in Zig.

```zig
const qoiz = @import("qoiz");

test "image init reader" {
    const file = try fs.cwd().openFile("image.qoi", .{});
    defer file.close();

    const image = try qoiz.Image(.rgba).initReader(
        testing.allocator,
        file.reader(),
    );
    defer image.deinit();
}
```
